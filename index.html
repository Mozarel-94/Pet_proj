Опыт

  • Руководство проектом в рамках госконтракта по разработке первой в России системы автоматизированного проектирования (САПР) СВЧ микроэлектроники (отечественный аналог Cadence AWR, Keysight ADS). Ссылка на публикацию
  • Постановка целей и задач для внутренней кросс-функциональной команды (разработчики, аналитики, СВЧ инженеры).
  • Координация работ подрядных организаций в роли ген. подрядчика (11 команд разработки из 4 компаний, суммарно более 150 человек): проведение совместных мероприятий по Scrum (демо, планирование) со всеми участниками кооперации, формирование календарного плана-графика работ, контроль выполнения работ.
  • Ведение задач проекта в Jira. Ведение базы знаний проекта в Confluence.
  • Проведение регулярных встреч с внутренней командой и командами подрядчиков.
  • Разработка технической документации для сдачи проекта в соответствии с ГОСТ (ЕСПД) и требованиями головного заказчика.
  • Организация проведения функциональных испытаний продукта.
  • Организация работ по интеграции наработок команд на едином стенде (CI/CD).

Стек: C++, QT, Python

Достижения:

  • В сжатые сроки (9 месяцев) успешно выполнены эскизный и технический проекты, созданы работающие технические прототипы САПР, не имеющей аналогов в России.
  • Обучил команды подрядных организаций работе по Agile (Scrum), модерировал проведение регулярных мероприятий (демо и планирование каждые 2 недели), выстроил процесс демонстрации результата головному заказчику на каждом демо.

Обязанности:

  • Разработка новых B2B продуктов с использованием ИИ (оптимизация транспортной логистики, прогноз выигрыша в тендерных закупках).
  • Поиск и проверка гипотез о применении ИИ в разных отраслях бизнеса (производство, розница, логистика).
  • Экономическое обоснование разработки продукта.
  • Набор команды разработки продукта с нуля.
  • Полный цикл разработки продукта с нуля: выявление потребности рынка, custdev, разработка прототипа, проведение пилотных проектов, вывод продукта в прод.
  • Проведение стендапов, демо, ретро.
  • Выступления с докладами про возможности применения ИИ в бизнесе.
  • Взаимодействие с университетами в рамках совместных проектов.

Достижения:

  • С нуля вывел продукт в рынок (AI-продукт по доставке светлых нефтепродуктов): набрал команду, сформировал бэклог продукта, провел пилотирование продукта.
  • Созданы работающие прототипы AI-продуктов по предсказанию успешности тендерных закупок, маршрутизации грузового транспорта, составлению оптимального распределения заказов на производстве.
